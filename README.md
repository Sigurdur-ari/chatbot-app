# AI Billing & Payment Support Chatbot

## Overview

This project is a lightweight AI-powered customer support chatbot prototype built to explore how conversational AI can improve customer experience in digital services.

The chatbot is focused specifically on billing and payment support scenarios for an Icelandic online web service, including:

- billing questions
- payment troubleshooting
- payment methods and billing information
- clarification and guidance for common payment-related issues

The primary goal of the project is not only AI integration itself, but also:

- conversation design
- response consistency
- structured support interactions
- safe fallback behavior
- user-focused communication

---

## Technical Details

The application is built with:

- React
- TypeScript
- Vite
- OpenAI API

The chatbot currently uses the `gpt-4o-mini` model for fast and cost-efficient responses during development.

A structured system prompt is used to:

- keep responses clear and concise
- ensure all responses are written in Icelandic
- guide the assistant toward billing and payment-related support scenarios
- encourage clarification through follow-up questions when needed
- avoid fabricated answers or unsupported claims
- redirect users toward customer support when issues fall outside the assistant's scope

The assistant is also instructed to provide structured responses using:

- explanations
- possible reasons
- next-step guidance

This helps create a more predictable and user-friendly support experience.
